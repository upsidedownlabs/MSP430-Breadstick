# MSP430 Breadstick

Upside Down Labs MSP430 Breadstick is a development board for ultra low power MSP430G2553 MCU from Texas Instruments. 

## Variant: Apple Sauce
Apple Sauce variant of MSP430 Breadstick comes with FT232RQ USB to UART bridge.
